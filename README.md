# Crud_Controle_Estoque
Sistema de gerenciamento de estoque de produtos, utilizando ReactJS e Node (Express). 
    Produto:{
                item:       descrição do produto, 
                valor:      preço do produto, 
                quantidade: número de itens no estoque
                }
                
    Métodos:
               Adicionar produtos,
               Listar produtos,
               Editar produto,
               Deletar produto,
               
}
